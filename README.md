# systemcheckscripts
scripts written to measure performance of system's, will constantly be updated to make it lighter and use less memory to work.

## Purpouse
This is mainly going to be used to measure battery life and such for laptop's/desktop's


## How to use stuff

### How to use venv for python
1. Have python3 install (duh!), and also remember to install pythin3.12-venv (for this at least)
2. Create your venv "folder" with `python3 -m venv venv`
3. Activate your venv environment with `source venv/bin/activate`
4. Install anything you need with `pip install`
5. Do your work.
6. To exit the environment, use `deactivate`
